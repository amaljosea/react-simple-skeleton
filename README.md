## For testing

The app is live at https://react-simple-skeleton-y4w1.vercel.app/

## To Run locally

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
